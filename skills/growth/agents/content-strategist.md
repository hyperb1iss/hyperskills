## Blog Post Structure

### SEO-Optimized Template

```markdown
# [Primary Keyword]: [Compelling Promise]

[Hook paragraph - address reader pain point directly]

[Preview what they'll learn - bullet points]

## [H2 with Secondary Keyword]

[Content section - 200-300 words]

### [H3 for subsection if needed]

[Supporting content with examples]

## [H2 with Related Keyword]

[Continue pattern...]

## Key Takeaways

- [Actionable insight 1]
- [Actionable insight 2]
- [Actionable insight 3]

## FAQ

### [Question with long-tail keyword?]
[Concise answer - great for featured snippets]

---
**Next Steps:** [CTA with internal link]
```

### SEO Checklist

```markdown
□ Primary keyword in title, H1, first 100 words
□ Secondary keywords in H2 headings
□ Meta description (150-160 chars) with keyword
□ URL slug contains primary keyword
□ 3-5 internal links to related content
□ 2-3 external links to authoritative sources
□ Alt text on all images
□ Minimum 1,500 words for pillar content
□ Table of contents for 2,000+ word posts
```

## Video Script Framework

### YouTube (8-15 min)

```markdown
## HOOK (0:00-0:30)
[Bold statement or question that creates curiosity]
"Did you know [surprising fact]?"

## INTRO (0:30-1:00)
[Brief context + promise of value]
"In this video, you'll learn exactly how to [outcome]"

## CONTENT SECTION 1 (1:00-4:00)
[First main point with examples]
- Visual: [B-roll/screen share description]
- Pattern interrupt at 2:00

## CONTENT SECTION 2 (4:00-7:00)
[Second main point]
- Visual: [Description]

## CONTENT SECTION 3 (7:00-10:00)
[Third main point]
- Visual: [Description]

## SUMMARY (10:00-11:00)
[Recap key points]
"So remember: [3 takeaways]"

## CTA (11:00-end)
[Subscribe, comment, related video]
"If this helped, hit subscribe and check out [related video]"
```

### TikTok/Reels (15-60 sec)

```markdown
## HOOK (0:00-0:03)
[Text on screen + immediate visual interest]
"Stop doing [common mistake]"

## CONTEXT (0:03-0:10)
[Quick setup - why this matters]

## VALUE (0:10-0:45)
[The actual tip/content]
- Quick cuts every 2-3 seconds
- Text overlays for key points

## CTA (0:45-0:60)
[Follow for more, save this, comment]
```

## Repurposing Workflow

```markdown
## From Blog Post to Multi-Platform

1. **Extract key quotes** → Twitter posts
2. **Summarize in bullets** → LinkedIn carousel
3. **Create visuals** → Instagram posts
4. **Record yourself reading** → TikTok/Reels
5. **Expand into script** → YouTube video
6. **Compile multiple posts** → Newsletter
7. **Combine related posts** → Lead magnet/ebook
```
